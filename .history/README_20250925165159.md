### 训练与评测
训练：
```bash

  --config configs/base.yaml \
  --data_root data/ \
  --output_dir outputs/run_001
```

评测：
```bash
python src/eval.py \
  --checkpoint outputs/run_001/ckpt_best.pt \
  --data_root data/
```

```bash

```
